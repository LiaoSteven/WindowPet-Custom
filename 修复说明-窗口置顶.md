# 🔧 修复说明：宠物被其他窗口遮挡

## 问题描述
在 macOS 上，宠物会被其他应用窗口遮挡，无法始终显示在最上层。

## 原因分析
- 使用 `maximized(true)` 在 macOS 上会导致窗口失去 `always-on-top` 属性
- macOS 的窗口管理机制与 Windows 不同

## 解决方案

### 修改内容：

1. **tauri.conf.json**
   - 窗口尺寸从 64x64 改为 3000x2000（覆盖大部分屏幕）
   - 添加 `focus: false`（避免窗口获取焦点）

2. **utils.rs**
   - 移除 `maximized(true)`
   - 使用固定大尺寸 `inner_size(3000.0, 2000.0)`
   - 添加 `position(0.0, 0.0)` 确保从屏幕左上角开始
   - 添加 `focused(false)` 确保不抢焦点

3. **main.rs**
   - 移除 `maximize()` 调用
   - 显式调用 `set_always_on_top(true)` 确保窗口置顶

## 效果
- ✅ 宠物始终显示在最上层
- ✅ 不会被其他应用窗口遮挡
- ✅ 窗口透明且可点击穿透
- ✅ 不会抢夺其他窗口的焦点
